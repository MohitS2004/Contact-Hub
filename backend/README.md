
# Contact Hub Backend

This is the backend service for the Contact Hub project, built with NestJS.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Copy `.env.example` to `.env` and update values as needed.
3. Run the development server:
   ```bash
   npm run start:dev
   ```

## Folders

- `docs/` – Documentation
- `uploads/` – File uploads
