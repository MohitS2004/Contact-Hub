# Contact Hub

A full-stack contact management application built with NestJS and Next.js.

## Structure

- `backend/` – NestJS backend service (PostgreSQL, TypeORM, JWT Auth)
- `frontend/` – Next.js frontend (TypeScript, Tailwind CSS)

## Features

- User authentication (Register/Login with JWT)
- Role-based access control (User/Admin)
- Contact management (CRUD operations)
- Responsive UI

Made with ❤️ for learning and productivity.
